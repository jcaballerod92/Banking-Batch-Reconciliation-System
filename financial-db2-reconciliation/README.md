# financial-db2-reconciliation

Mainframe-oriented reconciliation environment inspired by enterprise banking systems using COBOL, DB2 and JCL concepts.

This project is designed to demonstrate understanding of traditional financial batch processing environments still widely used in banking institutions.

---

# Main Components

- COBOL batch programs
- DB2 SQL scripts
- JCL job definitions
- Copybooks
- Batch execution documentation
- Reconciliation flow examples
