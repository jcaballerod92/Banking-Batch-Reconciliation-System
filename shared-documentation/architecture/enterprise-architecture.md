# Enterprise Architecture

<!-- Overview -->
This workspace is organized as a small enterprise-style portfolio that combines COBOL, DB2, JCL and modern Java services around a financial domain.

<!-- Structure -->
The goal is to present the projects as related layers rather than isolated exercises:
- mainframe batch processing
- DB2 data model and control
- Java API access layer
- shared documentation and portfolio narrative

<!-- Why it matters -->
This structure helps show continuity between legacy systems and modern backend development, which is especially relevant for banking and financial technology roles.
