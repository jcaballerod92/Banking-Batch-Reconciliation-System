# Financial Processing Flow

<!-- Flow -->
The reference flow for the portfolio is:
1. input validation
2. staging load
3. reconciliation
4. control reporting
5. inquiry/API access

<!-- Business meaning -->
This sequence reflects a realistic financial processing chain where raw movements are validated, reconciled against reference data and later queried by online services.

<!-- Outcome -->
The flow demonstrates traceability, data control and separation of responsibilities across the repository set.
